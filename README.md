## 个性化修改

修改标题名称之后，需要在head标签内的< style>标签内修改`h1::bofore`

~~~css
h1::before{
    content: 'KAYGB\'S HOME'; //修改此处
    position: absolute;
    color: #b2bec348;
    z-index: -1;
    left: 10px;
    top: 10px;
}

~~~
